# SFML_Lawn

### 这是一个用C++写的植物大战僵尸同人框架，致力于OpenGL，C++项目的学习和植物大战僵尸同人游戏的开发，支持植物大战僵尸一带的Reanim动画，支持多文件打包，支持纹理图集，使用自主开发的游戏框架SimpleGameFramework，支持3D模型渲染（底层OpenGL）

# How to build
使用vcpkg安装项目依赖项，x86平台，编译，运行，在生成exe的同级目录下放入资源文件，即可正常运行
